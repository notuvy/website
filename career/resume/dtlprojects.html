Extracurricular Software Projects

Besides the software development I do at work, I also work on several projects on my own. Some of them are open source projects, and many are just for my own use. Although there may be publicly available software packages that do some of the things these do, I specifically wanted to develop these on my own. Sometimes it is to ensure that a particular feature is implemented. Sometimes it is to get practice with particular technologies (e.g. image processing, XML, etc.). Always there is an opportunity to enhance my design and refactoring skills.

Open Source

Fsgrep

I am the author and sole developer of the open source project Fsgrep, a multiple file, regular expression based search tool. The source code is distributed with the download, so it is a concrete example of my design skills and coding style.

Follow

I am a contributing developer on the open source project Follow, a graphical multiple log file monitoring tool.

NOTUVY singleapp

I am the author and sole developer of the open source project NOTUVY singleapp, which provides a simple mechanism to make Java desktop applications singleton instances. It enforces that only one instance of an application (or one per user) is allowed to run on the system. The developer can easily configure whether a new instance will defer to an existing instance, or preempt it.

NOTUVY utils

I am the author and sole developer of the open source project NOTUVY util, a Java library of utilities to facilitate rapid application development.

NOTUVY regex

I am the author and sole developer of the open source project NOTUVY regex, a Java library providing frameworks for simple yet powerful regular expression usage.

Pesonal

jpodc

A podcast portfolio and download manager. It lets you organize all the podcast you subscribe to in a logical structure where you control the download schedule and it manages the downloades audio files.

streamrecorder

Like a DVR for streaming audio, streamrecorder allows you to do scheduled downloads of streaming media. The resulting files are of the fixed (time) length as you define. The results are published as RSS so you can consume them as private podcasts.

trip

A tool to publish trip itineraries with current flight status.

jviewer

An image viewer.

ftpfile

An FTP file publish/retrieval tool.

recipe

A tool that processes my (filesystem/XML) database of recipes and indexes and publishes them as HTML. (The results are viewable online.)

geotools

A tool to manage geoprahical information as GPX files. Much of the logic allow sharing of data with Google Maps, and allows transformed data to be entered into a GPS device.

picmgr

A tool construct and publish digital photo albums and web logs. (This has become obsolete with the public release of Picasa and similar tools.)
