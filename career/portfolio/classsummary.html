Sample Project: Description of Classes

The table below give a brief description of the classes. Click on the class name to view the source code.

ClassDescription
HtmlCal The main driver for the logic. The main() provides command line execution, and its public API can be used by other programs.
Range Represent a contiguous range of dates. Implemented by just storing the start and end dates of the range. Any number of additional dates can be added. If a new date is within the range, then the addition is a noop. If a new date is outside the range, then the range is expanded.
Month A composition of Week objects.
Week A composition of DayOfMonth objects.
DayOfMonth The representation of a day of the month. It contains the actual object representing that date, along with logic that determines how it should eventually be rendered.
DayOfWeek An enumeration of the representation of the day of the week. The underlying id is the same value used in Calendar.
DayDate The object representing the actual day as a time object. We could not simply use Date or Calendar because neither give sufficient functionality in a single object. This does by combining features from both. All the date/time manipulation logic is isolated to this class.
Printable An object that can print itself as HTML.
Markable An object that can mark itself as being part of a Range.
