SnipFolio

SnipFolio moves the copy-and-paste paradigm to the cloud. SnipFolio integrates with the system clipboard of your multiple environments (work computer, home computer, smartphone, etc.) so that as soon as a snippet is copied on one, it is available for paste on the others.

Even if you use it on a single computer, SnipFolio can be highly useful. It maintains a history of clipboard contents so you do not lose the previous contents when you copy a snippet.1 It adds features to the basic clipboard such as:

  • Grouping of snippets in persistent lists
  • Powerful transform of contents by regular expression2
  • Simple editing of contents3

1 Have you ever copied multiple fields (like name, address, phone number) from a document to a web form? It is much faster and easier (and less error prone) to do multiple copies from the source document followed by multiple pastes in the target location than it is to keep switching between documents to do multiple copy-and-paste. You need the history of multiple copied snippets to do this.

2 Say you want to copy names from one document to another, but the source document has them in the form "Last, First" but the target document has them in the form "First Last". You could define a transform in SnipFolio that automatically switches the order so when you copy the name, it enters the clipboard as "First Last". You can then simply cut and paste with no manual editing in either document. This can be a huge time saver (and error preventer).

3 Have you ever copied something that you pasted but then had to make hand modifications to? If you are pasting and editing in multiple places, this can get tedious. SnipFolio lets you edit the snippet while it is in the clipboard, so you can edit it once then paste it multiple times.
